    Python API

    Run#

    Start up a Lassoo server using uvicorn, using the data and umap model passed in as arguments to serve requests.

    def run(
    data: pandas.DataFrame,
    out_path: str,
    umap: "umap.UMAP" = None,
    db: sqlite_utils.Database = None,
    text_column: str = "text",
    label_column: str = None,
    search_columns: List[str] = None,
)

    Arguments#

    • data - A pandas DataFrame, containing a column of text data and an optional column of labels.
    • out_path - A filepath to save the data DataFrame with the updated user provided annotations.
    • umap - A UMAP model to use, created from the data DataFrame.
    • db - A sqlite database containing a "data" table with full text search enabled. If no db is passed, text search is disabled.
    • text_column - The string name of the text column to embed. Default: 'text'.
    • label_column - The string name of the label column, if there are existing labels.
    • search_columns - A list of column names to index. Redundant if a db is passed.

    Embed#

    Lassoo's embed function is a shortcut for creating a UMAP model from a Pandas DataFrame.

    Note

    If you are familiar with UMAP and have a custom method for producing your embeddings and UMAP models, there's no need to use this function. lassoo.run will work with any UMAP model and metadata. 

    def embed(
    data: pandas.DataFrame,
    text_column: str = "text",
    label_column: str = None,
    encoder: str = None,
    out_path: str = None,
    min_dist: float = 0.00,
    n_neighbors: int = 5,
) -> "umap.UMAP":

    Arguments#

    • data - A pandas DataFrame, containing a column of text data for embedding.
    • text_column - The string name of the text column to embed. Default: 'text'.
    • label_column - The string name of the label column, if there are existing labels.
    • encoder - The name of a valid encoder from the sentence-transformers library. By default, a TF-IDF encoder is used.
    • out_path - A filepath to save the UMAP model to, if provided.
    • min_dist - The minimum similarity distance for UMAP.
    • n_neighbors - The number of nearest neighbors for UMAP.

    Returns#

    A UMAP model.

    Index#

    Lassoo's index function creates a sqlite database used for searching over your data. This database can be passed to lassoo.run as an option, allowing selection of data via search as well as using the visual selection tool.

    def index(
    data: pandas.DataFrame,
    search_columns: List[str],
    db_path: str = None
) -> sqlite_utils.Database:

    Arguments#

    • data - A pandas DataFrame, containing a column of text data for embedding.
    • search_columns - A list of column names to index. Typically this would be the same as the text_column, but could also be additional columns.
    • db_path - A path to save the created search db, if provided.

    Returns#

    A sqlite_utils.Database containg a table called data, which has full text search enabled.

    Info

    sqlite_utils is a convenience library for interacting with sqlite databases from Python.
