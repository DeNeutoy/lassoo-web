    Lassoo Command Line Usage#

    Lassoo also contains a command line tool to analyse data from a terminal.

    Run#

    Usage: lassoo run [OPTIONS] DATA_PATH OUT_PATH

    View data in the Lassoo interface.

    Arguments:
  DATA_PATH  The path to the data.  [required]
  OUT_PATH   The output path to save the updated/labelled data.  [required]

Options:
  --umap-path TEXT               The path to a serialised umap joblib file.
  --db-path TEXT                 The path to a sqlite db file created by
                                 `index`.

  --text-column TEXT             The name of the text column in the data.
                                 [default: text]

  --label-column TEXT            The name of the label column in the data.
  --search-columns TEXT          The name of the columns to index for
                                 searching in the data.

  --format [csv|tsv|jsonl|text]  The file format the data is stored in.
  --help                         Show this message and exit.

    Embed#

    Usage: lassoo embed [OPTIONS] DATA_PATH OUT_PATH

    Embed data using an encoder and project it to 3D using UMAP. By default, a tf-idf encoder is used, but more powerful encoders can be used from sentence-transformers. This saves a UMAP joblib file which can be passed to lassoo run using the --umap-path argument.

    Arguments:
  DATA_PATH  The path to the text file.  [required]
  OUT_PATH   The output path for saving the UMAP model.  [required]

Options:
  --text-column TEXT             The name of the text column in the data.
                                 [default: text]

  --label-column TEXT            The name of the label column in the data.
  --encoder TEXT                 The sentence-transformer model to use.
  --min-dist FLOAT               The UMAP min_dist parameter.  [default: 0.0]
  --n-neighbors INTEGER          The UMAP n_neighbors parameter.  [default: 5]
  --format [csv|tsv|jsonl|text]  The file format the data is stored in.
  --help                         Show this message and exit.

    Index#

    Usage: lassoo index [OPTIONS] DATA_PATH DB_PATH SEARCH_COLUMNS...

    Index a dataset for full-text search. This generates a sqlite db of a dataset, which will allow you to search the dataset in the interface. This is provided as a separate command so that the index creation can be cached, and passed to lassoo run using the --db-path argument.

    Arguments:
  DATA_PATH          The path to the data (tsv or csv).  [required]
  DB_PATH            The path to save the database to.  [required]
  SEARCH_COLUMNS...  The name of the columns to index for searching in the
                     data.  [required]

Options:
  --format [csv|tsv|jsonl|text]  The file format the data is stored in.
  --help                         Show this message and exit.
